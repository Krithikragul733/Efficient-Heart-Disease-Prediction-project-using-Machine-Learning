# ❤️ Heart Disease Prediction using Explainable Ensemble Learning

## Overview
This project predicts heart disease using Ensemble Machine Learning techniques such as Random Forest and XGBoost. Explainable AI methods are integrated to improve transparency and interpretability.

## Features
- Heart Disease Prediction
- Ensemble Learning
- Explainable AI (SHAP)
- Streamlit Web Application
- Risk Assessment

## Technologies Used
- Python
- Scikit-Learn
- XGBoost
- SHAP
- Streamlit
- Pandas
- NumPy

## Input Parameters
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- CA
- Thal

## Run the Application

```bash
pip install -r requirements.txt
streamlit run app.py
```
